# products_storage
В вашем распоряжении находится склад, обладающий функционалом для управления товарными запасами. Данный склад предоставляет возможность динамического добавления и удаления продуктов, а также настройки их атрибутов, таких как наименование, срок годности и другие релевантные параметры.
